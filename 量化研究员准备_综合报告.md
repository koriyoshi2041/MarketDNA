## 一、五人共识

### 1. 你的核心卖点不是"懂金融"，是"能把AI工程化落地到金融问题"

所有角色都独立得出同一结论：不要把自己包装成"金融专家"，要包装成 **"能把前沿AI方法工程化落地到金融问题的人"**。这是你的真正赛道。

### 2. 工程能力远超同龄候选人，这是稀缺组合

- Python + Rust + Go + TypeScript，有Docker/FastAPI/全栈经验
- 50+ PRs across 23 repos，能读别人的代码、能理解系统、能交付
- 在量化行业，"能写生产代码的研究员"极其稀缺

### 3. 金融素养为零是致命短板，没有捷径

- 不是"弱"，是"零"
- 不知道Sharpe Ratio、Alpha/Beta、前瞻偏差（look-ahead bias）
- 面试中一个简单的金融问题就会暴露
- 必须系统性学习3-6个月

### 4. 数学竞赛背景缺失是最大的简历关卡

- 头部量化（幻方/九坤/Jane Street）筛简历第一眼看：数学竞赛（CMO/丘赛）、信息学竞赛（NOI/ICPC）
- 你一个都没有
- 只能用项目和实战经历弥补

### 5. 时间是你最大的资产

- 大二开始准备，时间窗口非常充裕
- 大多数人大三才开始想量化
- 第一份实习是最难拿的，第二份开始就容易了

### 项目一：MarketDNA — 金融时间序列统计指纹提取（6-8周，★★☆）

**目标**：系统化检测金融数据中违背经典假设的统计异象，建立统计推断基础

**你将学到**：
- 收益率分布建模（Student-t、混合高斯）
- 时间序列计量经济学（ADF/KPSS、ARIMA、GARCH族）
- 波动率建模（Realized Vol、EGARCH杠杆效应）
- 相关性矩阵去噪（Random Matrix Theory — Marchenko-Pastur定律）
- 协整检验与配对交易原型

**技术栈**：
- 数据：yfinance (美股20年日频) + FRED (宏观) + Binance (高频练习)
- 核心：Python (statsmodels, arch, scipy, hmmlearn)
- 可视化：Plotly/Streamlit交互式dashboard

**关键步骤**：
1. 数据管道搭建（Parquet列式存储）
2. 正态幻觉破除（QQ-plot、Jarque-Bera检验、尾部厚度量化）
3. GARCH波动率建模（理解杠杆效应：坏消息比好消息引起更大波动）
4. Random Matrix Theory相关性去噪（500只股票的样本相关矩阵80%+特征值是噪声）
5. Engle-Granger协整检验 + Ornstein-Uhlenbeck过程建模
6. HMM Regime检测（高波/低波状态自动发现）
7. 统计异象信号化 + 多重检验校正（BH-FDR）

**简历一句话**：
> Built a statistical anomaly detection framework implementing GARCH-family models, Johansen cointegration tests, and RMT-based correlation denoising across 500+ equities, identifying regime-dependent signals with FDR-corrected significance.

### 项目二：AlphaForge — 多因子Alpha研究与组合优化引擎（8-10周，★★★）

**目标**：构建完整的因子投资研究平台，从因子构建到组合优化的全流程

**你将学到**：
- 因子投资理论（Fama-French、Barra风险模型）
- 截面回归（Fama-MacBeth回归、IC/IR分析）
- 组合优化（Markowitz、Black-Litterman、Risk Parity）
- 交易成本建模（滑点、市场冲击、换手率约束）
- Alpha衰减分析

**技术栈**：
- 数据：yfinance + SimFin (基本面) + Kenneth French Library
- 核心：Python (cvxpy, statsmodels, sklearn), Polars
- 回测：自建事件驱动回测引擎

**关键步骤**：
1. 因子动物园构建（15-20个经典因子 + 行业/市值中性化）
2. 单因子检验流水线（IC分析、分层回测、Fama-MacBeth、Newey-West t统计量）
3. 因子正交化（Löwdin正交化 — 业界核心技术）
4. Alpha组合模型（等权/IC加权/Ridge回归组合）
5. CVXPY组合优化（带约束：换手率、行业中性、因子暴露）
6. Black-Litterman模型（将alpha信号作为"view"输入）
7. 交易成本建模（固定+线性+非线性冲击）
8. 归因分析（Brinson归因 + 因子归因）

**核心优势发挥**：你的L-BFGS-B/凸优化经验在组合优化模块直接可用

**简历一句话**：
> Developed a multi-factor alpha platform with 20+ factors, Fama-MacBeth validation, Löwdin orthogonalization, and CVXPY portfolio optimization (MVO, Risk Parity, Black-Litterman) with realistic transaction cost modeling, achieving out-of-sample IR of X.XX.

### 项目三：RegimeNet — 基于深度学习的市场Regime自适应交易系统（10-12周，★★★★）

**目标**：解决ML在金融中的三大魔鬼：过拟合、非平稳性、regime change

- Triple Barrier标记法 + Meta-Labeling（Lopez de Prado方法）

- 在线学习与模型衰减监控

**技术栈**：

**核心创新**：Regime-Conditioned预测架构
```
输入特征 → 共享特征提取 → Regime分类器 → p(regime|x)
                                ↓
                    Regime-specific预测头 → 加权输出 + 校准置信度
```

**NCAA经验直接对接**：Brier Score概率校准 → Meta-Labeling置信度评估

**简历一句话**：

### 项目四：MicroPulse — RL做市与最优执行系统（10-12周，★★★★★）

**目标**：用深度RL构建做市策略和最优执行算法，展示市场微观结构理解

**你将学到**：
- 市场微观结构（订单簿动力学、Kyle/Glosten-Milgrom模型）
- 做市策略（Avellaneda-Stoikov模型、库存风险管理）
- 最优执行（Almgren-Chriss框架、Implementation Shortfall）
- 深度RL在金融中的应用（Constrained RL、Reward Shaping）

**技术栈**：
- 数据：LOBSTER (纳斯达克L3订单簿) 或 Binance WebSocket
- RL：PyTorch (PPO/SAC) + 自定义Gym环境
- **Rust加速**：用Rust写订单簿撮合引擎（<1μs/order），通过PyO3绑定到Python
- 理论Baseline：Avellaneda-Stoikov解析解

**三层LOB模拟器**：
1. History Replay — 真实订单流回放
2. Statistical Model — Hawkes过程模拟订单到达
3. Agent-Based — 多agent环境（趋势跟踪者+均值回复者+噪声交易者）

**核心RL设计**：
- State：订单簿快照(CNN处理) + 库存 + PnL + 时间
- Action：bid/ask的offset和size
- Reward：realized PnL - γ×inventory² - 安全约束违约罚金
- Trick：用A-S解析解做reward shaping加速收敛

**简历一句话**：
> Built a deep RL market-making system on L2 order book data with a 3-tier LOB simulator, Avellaneda-Stoikov reward shaping, constrained PPO/SAC, and a Rust matching engine (<1μs latency via PyO3).
